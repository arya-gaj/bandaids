# bandaids.

A modern, minimalist landing page for bandaids - your job search, made easier.

## Overview

bandaids is a clean, single-page landing page built with Next.js, featuring smooth animations and a glassmorphism design aesthetic. The page showcases the upcoming platform that will help job seekers find opportunities that truly fit.

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd 2025_iiit_build2break
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open [http://localhost:80](http://localhost:80) in your browser

### Build for Production

```bash
npm run build
npm start
```
